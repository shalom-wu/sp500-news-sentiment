# Power BI Package

No `.pbix` is included for this project yet. Power BI Desktop is installed, but creating and saving a new `.pbix` requires a manual GUI step; I did not create a placeholder file.

Included:

| File/folder | Purpose |
|---|---|
| `../data/powerbi/` | CSV inputs generated by `python scripts/run_sql.py`. |
| `dashboard_brief.md` | Dashboard plan. |
| `data_model.md` | Tables and relationships. |
| `dax_measures.md` | Measures to create. |
| `manual_build_instructions.md` | Rebuild guide. |
| `refresh_instructions.md` | Refresh workflow. |
| `screenshots/page*_mockup.png` | Static mockups generated from included data, not actual Desktop screenshots. |

The dashboard should emphasize the main conclusion: sentiment is better as market context than as a standalone next-day prediction signal.
